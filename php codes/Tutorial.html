*이미지 업로드 가이드*

1. PHP Admin (DB정보 볼 수 있어요)
2. Web FTP(서버 폴더 www폴더 안에 파일들 있어요)

3. 이미지 업로드 URL

 3-1). 파일선택으로 사진 파일 등록

 3-2). 빈칸에 태그 입력 (+로 태그 연결), 온도 구간 선택 (순서대로 TEMPERATURE 테이블 ID 1,2,3,4로 들어감)

 3-3). POST 버튼 클릭 => 서버에 파일 업로드(www/images폴더에 들어감) & DB에 파일 경로와 함께 이미지 정보 저장 (IMAGE 테이블)

 3-4). 성공적으로 저장된 경우 이미지와 함께 태그, 온도 ID가 출력됨

  * 주의

   - 이미 동일한 이름의 image파일이 존재할 경우 처리를 안했습니다. 이름 겹치지 않게 주의해주세요.

   - 사진 파일 이름은 한글인 경우 깨져요 (경로설정이 달라져요)
